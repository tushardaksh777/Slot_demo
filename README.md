# Slot_demo

# Unity Game Assignment

This repository contains a Unity game developed using Unity Engine and C#.  
It was created as part of an interview assignment to demonstrate gameplay logic, code structure, and Unity fundamentals.

---

## Tech Stack

- Engine: Unity
- Language: C#
- Platform: PC (Unity Editor)
- Unity Version: Unity 6.3.6f1

---

## Project Structure

Assets/
- Scripts/   → Gameplay and logic scripts
- Scenes/    → Game scenes
- Prefabs/   → Reusable game objects
- Art/       → Sprites / models
- Audio/     → Sound effects / music

---

## How to Run the Game

1. Install Unity Hub  
2. Install Unity **6.3.6f1**  
3. Clone this repository:  
   git clone https://github.com/tushardaksh777/Slot_demo.git  
4. Open Unity Hub  
5. Click **Open Project**  
6. Select the cloned project folder  
7. Open the main scene from `Assets/Scenes`  
8. Press the **Play** button in the Unity Editor  

---

## Implementation Overview

- Gameplay logic written in C#
- MonoBehaviour-based architecture
- Modular and readable code
- Focus on stability and maintainability
